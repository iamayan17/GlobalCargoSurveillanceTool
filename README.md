# Global Cargo Surveillance Tool

A Django-based web application designed to monitor and manage cargo-related data efficiently.

## Features
- User authentication
- Cargo data monitoring
- Admin panel support
- Structured MVC architecture

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite3

## Project Structure
- Project folder: settings and configurations
- App folder: business logic
- Templates: HTML files
- Static: CSS and static assets

## How to Run
1. Install Python and Django
2. Navigate to project folder
3. Run: python manage.py runserver
4. Open: http://127.0.0.1:8000/
